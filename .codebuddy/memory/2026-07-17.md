# 2026-07-17 工作记录

## 新增：管理员改密码功能（用户存储改为数据库）
- 背景：原登录仅基于 `appsettings.json` 的 `AppUsers`，密码无法持久修改。
- 改动：
  - `Infrastructure/Entities/UserEntity.cs`：增加 UserName/Password/Role 列。
  - `Core/Users/Entities/User.cs`：领域增加 Password/Role，`GetClaims()` 改用真实 Role（默认 admin）。
  - `Core/Users/IUserRepository.cs`、`Infrastructure/Reposities/UserRepository.cs`：新增 `GetByUserNameAsync`。
  - `Application/Users/UserAppService.cs`：`LoginAsync` 改为查库，库中无用户且配置匹配时自动 seed 入库（兼容旧配置）。
  - 新增 `WebApi/Features/User/ChangePassword`（端点 `POST /auth/change-password`）：需登录；管理员可改他人密码，普通用户改自己需校验旧密码。
  - `WebApi/Features/User/Login/Endpoint.cs`：JWT 携带真实角色。
  - 前端：`api/user.ts` 增加 `changePassword`；新增 `components/common/ChangePasswordDialog.vue`；`layouts/MainLayout.vue` 用户菜单加“修改密码”入口。
- 验证：`dotnet build` 后端 0 错误。前端无 node_modules，未做 type-check。

## 加固：表新增记录 InsertData
- `WebApi/Features/Table/InsertData/Endpoint.cs`：重写 `GenerateInsertSql`，转义表名/列名，按类型处理数字/NULL/布尔，避免原实现强制加引号与 SQL 注入问题。
